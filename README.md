# 🖥️ Contador com React + Redux

Este projeto é um contador simples desenvolvido com **React** e **Redux**, criado para praticar:

- Manipulação de estado global com Redux
- Hooks do React (`useSelector`, `useDispatch`)
- Eventos em React
- Estruturação de projetos front-end modernos

---

## 🚀 Funcionalidades

- Incrementar o valor do contador
- Decrementar o valor do contador
- Atualização em tempo real através do estado global
- Interface simples e funcional

---

## 🛠️ Tecnologias utilizadas

- **React**  
- **Redux** / **React-Redux**  
- **JavaScript (ES6)**  
- **HTML5 / CSS3**
